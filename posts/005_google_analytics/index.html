How to use google analytics in hugo website?

Where to put the google analytics codes?

February 15, 2022

  • google analytics

On Hugo website, by searching Google Analytics will find configure google analytics codes like these:

googleAnalytics = 'UA-PROPERTY_ID'

{{ template "_internal/google_analytics.html" . }}

Put the first code at config.toml file.
Put outside the [params], not in it. It’ll look like this:

baseURL = "https://KPW10452025.github.io"
title = "Webb"
...
googleAnalytics = 'UA-220340577-1'

[params]
  footer = "Love & Peace"
  description = "Hi! I'm Wdbb Kuo. This is my first personal website."
  ...

Put the second code at head.html file. ( themes/layouts/partials/head.html )
Just put it inside the head tags.

<head>
    ...
    ...
    {{ template "_internal/google_analytics.html" . }}
</head>
