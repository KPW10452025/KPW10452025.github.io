JS 課程015 算數運算符

By Webb 2022-03-01

本篇要介紹的是，各種常用算數運算符在 JS 中的效果

何謂運算符？

運算符：亦可稱作操作符，可對一個或多個值進行運算

前面課程提到的 typeof 就是一個運算符

var a = 123;
var result = typeof a;
console.log(result); // number 
console.log(typeof result); // string 由此可知，typeof 的返回值是字符串

算術運算符 + - * / %

  • 加法

當使用算術運算符對非數字的值進行運算時，會自動將這些值轉換為數字再進行運算

var a = 10;
var result = a + 2;
console.log(result); // 12

var result = true + 1;
console.log(result); // 2 
// true = 1

var result = true +false;
console.log(result); // 1
// false = 0

var result = 3 + null
console.log(result); // 3
// null = 0

var result = 123 + NaN;
console.log(result); // NaN
// 任何值與 NaN 進行運算都會得到 NaN

任何值與字符串做 + 運算，都會先轉換成字符串後拼接

var result = "123" + "456";
console.log(result); // 123456

var result = "123" + 123
console.log(result); // 123123

var result = "號碼" + 123;
console.log(result); // 號碼123

var result = "號碼" + true;
console.log(result); // 號碼true

var result = "號碼" + NaN;
console.log(result); // 號碼NaN
// 結果並不是 NaN

可以運用此特性，將任意類型數據轉換為字符串 String

var result = 333;
result = result + "";
console.log(typeof result); // string
console.log(result); // 333

這是一種「隱式」的類型轉換
實際上，系統有自動調用 String() 函數
result = result + ""
result = String(result)
以上兩者本質一樣，但「隱式」轉換會表示的更簡單一些

  • 減法

除了 NaN 與 undefined 以外，都會強制轉換為數字後運算

var c = 100 - 1;
console.log(c); // 99

var c = 100 - "3";
console.log(c); // 97

var c = 100 - true;
console.log(c); // 99

var c = 200 - false;
console.log(c); // 200

var c = 100 - NaN;
console.log(c); // NaN

var c = 100 - undefined;
console.log(c); // NaN

var c = "100" - "3";
console.log(c); // 97

var c = "100" - true;
console.log(c); // 99

var c = true - "3";
console.log(c); // -3

var c = "100" - "apple";
console.log(c); // NaN
  • 乘法

除了 NaN 與 undefined 以外，都會強制轉換為數字後運算

var c = 100 * 3;
console.log(c); // 300

var c = 100 * "3";
console.log(c); // 300

var c = 100 * false;
console.log(c); // 0

var c = 100 * NaN;
console.log(c); // NaN

var c = 100 * undefined;
console.log(c); // NaN

var c = "apple" * true;
console.log(c) // NaN

var c = "apple" * 3;
console.log(c) // NaN
  • 除法

除了 NaN 與 undefined 以外，都會強制轉換為數字後運算

var c = 20 / 5;
console.log(c); // 4

var c = 20 / "5";
console.log(c); // 4

var c = 20 / true;
console.log(c); // 20

var c = 20 / NaN;
console.log(c); // NaN

var c = 20 / undefined;
console.log(c); // NaN

由上面所有的例子可以發現 - * / 都會強制轉換成數字 number 進行運算
可以運用此特性，將任意類型數據轉換為數字 number
原理和 Number() 函數一樣，為「隱式」類型轉換

ar c = "123" - 0;
console.log(typeof c); // number
console.log(c); // 123

var c = "123" * 1;
console.log(typeof c); // number
console.log(c); // 123

var c = "123" / 1;
console.log(typeof c); // number
console.log(c); // 123
  • 取模運算 modulus
var c = 9 % 3;
console.log(c); // 0 

var c = 9 % 5;
console.log(c); // 4  

var c = 9 % true;
console.log(c); // 0  

var c = 9 % "6";
console.log(c); // 3
