JS 課程012 強制類型轉換之 Number() 函數與 parseInt() 函數

By Webb 2022-02-28

本篇要介紹的是，將其他的數據類型轉會為數字 Number

其方法只有一種：

  1. Number() 函數

注意！！！：並不存在 toNumber() 方法！！！

  1. 特別介紹：parseInt() 函數、parseFloat() 函數

Number()函數

  1. Number(字符串)：
      如果是純數字的字符串，則直接將其轉換為數字
      如果字符串中非數字內容，則轉換為 NaN
      如果字符串中沒有字符或是空格，則會轉換為 0
  2. Number(布林值)：
      如果是 true，則轉換為 1
      如果是 false，則轉換為 0
  3. Number(空值)：會轉換為 0
  4. Number(未定義)：會轉換為 NaN
  1. 字符串轉數字
var a = "123"
console.log(typeof a); // string
console.log(a); // 123
a = Number(a);
console.log(typeof a); // number 轉換成功
console.log(a); // 123

var a = "apple"
console.log(typeof a); // string
console.log(a); // apple
a = Number(a);
console.log(typeof a); // number 轉換成功
console.log(a); // NaN 非數字內容，轉換為 NaN

var a = ""
console.log(typeof a); // string
console.log(a); // 
a = Number(a);
console.log(typeof a); // number 轉換成功
console.log(a); // 0 沒有字符，轉換為 0

var a = "     "
console.log(typeof a); // string
console.log(a); // 
a = Number(a);
console.log(typeof a); // number 轉換成功
console.log(a); // 0 空格，轉換為 0
  1. 布林值轉數字
var a = true;
console.log(typeof a); // boolean
console.log(a); // true
a = Number(a);
console.log(typeof a); // number
console.log(a); // 1

var a = false;
console.log(typeof a); // boolean
console.log(a); // false
a = Number(a);
console.log(typeof a); // number
console.log(a); // 0
  1. 空值轉數字
var a = null;
console.log(typeof a); // object
console.log(a); // null
a = Number(a);
console.log(typeof a); // number
console.log(a); // 0
  1. 未定義轉數字
var a = undefined;
console.log(typeof a); // undefined
console.log(a); // undefined
a = Number(a);
console.log(typeof a); // number
console.log(a); // NaN

假設情況如下述：我們從CSS中擷取到一個字符串：200px，並且想把這像素加上 20px 此時要怎麼做？

var a = "200px";
a = Number(a);
console.log(typeof a); // number
console.log(a); // NaN

若直接使用 Number() 則會得不到我們想要的結果
因為字串中有非數字 px 所以無法使用 Number() 做轉換

parseInt()函數：將一個字符串中有效的內容取出來，並轉換為整數

parseFloat()函數：將一個字符串中有效的內容取出來，並轉換為浮點數

var a = "200px";
a = parseInt(a);
console.log(typeof a); // number
console.log(a); // 200

由上述可知，藉由 parseInt() 成功擷取到 200 是數字型態

parseInt() 函數與 parseFloat() 函數之特性解說：

  1. parseInt 只會由左往右取，當碰到無效內容就會停止
var a = "2022/02/28";
a = parseInt(a);
console.log(typeof a); // number
console.log(a); // 2022

var a = "47.22353px";
a = parseInt(a);
console.log(typeof a); // number
console.log(a); // 47
  1. parseFloat 只會由左往右取，當碰到無效內容就會停止
var a = "2022/02/28";
a = parseFloat(a);
console.log(typeof a); // number
console.log(a); // 2022

var a = "47.22353px";
a = parseFloat(a);
console.log(typeof a); // number
console.log(a); // 47.22353
  1. 特別討論：用 parseInt() 轉換布林值
var a = true;
a = parseInt(a);
console.log(typeof a); // number
console.log(a); // NaN
// 如果 parseInt() 對非字符串做轉換時，會先將其轉換為字符串後再擷取
