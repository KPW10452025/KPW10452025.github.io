JS 課程023 相等運算符

By Webb 2022-03-04
  1. == 相等（只比較值，不比較類型，類型不同時會自動轉換類型）
  2. != 不相等（只比較值，不比較類型，類型不同時會自動轉換類型）
  3. === 全相等（比較值，也比較類型）
  4. !== 不全相等（比較值，也比較類型）
  • 相等運算符 ==

使用 == 來做相等運算，比較兩個值是否相等

在 JS 中 == 並不會比較類型，如果類型不同，系統會自動轉換並比較

如果相等會回傳 true

如果不相等會回傳 false

var a = 10;
result = a == 10;
console.log(result) // true

var a = 11;
console.log(a == 11); // true

注意：系統並不一定是轉成數字 Number 進行比較

console.log("1" == 1); // true

console.log("apple" == 1); // false

console.log(true == 1); // true

console.log(true == "hello"); // false

console.log(null == 0); // false

undefined 衍生自 null，所以這兩個值做相等判斷時會出現 true

console.log(null == undefined); // true

NaN不和任何值相等，包括自己

console.log(NaN == 0); // false

console.log(NaN == "1"); // false

console.log(NaN == false); // false

console.log(NaN == null); // false

console.log(NaN == undefined); // false

console.log(NaN == NaN); // false

如果今天要判斷一個變數是否為 NaN 怎麼辦？

var b = NaN;
console.log(b == NaN); // false 此法無法判斷

可以通過 isNaN() 函數來判斷一個值是否是 NaN

var c = NaN;
console.log(isNaN(c)) // true 判斷成功
  • 不相等運算符 !=

使用 != 來做不相等運算

用來比較兩個值是否不相等

可以不同類型進行比較

如果不相等會回傳 true

如果相等會回傳 false

console.log(10 != 5); // true

console.log(10 != 10); // false

當使用兩種不同類型時，會自動轉換成相同類型進行比較

注意：系統並不一定是轉成數字 Number 進行比較

console.log("apple" != 5); // true

console.log(1 != "1"); // false
  • 全等於運算符

使用 === 來做全等於運算符

與相等 == 不一樣的是，比較時不會做類型轉換

用來比較兩個值是否相等，並且連類型皆相等

如果全相等，值相等且類型相等，會回傳 true

其餘回傳 false

console.log("apple" === "apple"); // true

console.log(1 === 1); // true

console.log(true == 1); // true
console.log(true === 1); // false

console.log(null == undefined); // true
console.log(null === undefined); // false
  • 不全等於運算符

使用 !== 來做不全等於運算符

與相等 != 不一樣的是，比較時不會做類型轉換

console.log(true != 1) // false
console.log(true !== 1) // true

console.log(null != undefined) // false
console.log(null !== undefined) // true
