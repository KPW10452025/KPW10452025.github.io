JS 課程003 JS 代碼編寫位置

By Webb 2022-02-18
  • attribute 屬性
  • execute 執行
  • practical application 實際應用
  • maintenance 維護
  • couple 偶合
  • quote 引用
  • caching mechanism 緩存機制
  1. We can code the JS in the attributes of onclick.
    JS code will be executed when we clicking the button on the website.
<body>
  <button onclick="alert('Hello there!')">Click me!</button>
</body>
  1. We can also code the JS in the a tag.
    JS code will be executed when we clicking the hyperlink.
<body>
  <a href="javascript:alert('Look over there!')">Click me!</a>
</body>
  1. Here is a practical application for JS in a hyperlink.
    When we code an a tag and we don’t want it to connect any urls, we can use 【 javascript:; 】.
<body>
  <a href="javascript:;">Click me!</a>
</body>
  1. Although JS can be written in attributes, it is easy to couple attributes with structures.
    Because maintenance is difficult, we don’t recommend.
    By using the script tag to code JS in head tag is better than on the attributes.
<head>
  <script type="text/javascript">
    alert("This is an alert!")
  </script>
</head>
  1. The best way to code JS is creating a .js file.
    And using script tags and src in .html file to link the .js file.
    The benefits of writing JS code in the .js file is we can quote this file to different .html file. In this way, we can also take advantage of the browser’s caching mechanism.
<head>
  <script src="/example.js"></script>
</head>
