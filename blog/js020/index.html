JS 課程020 非布林值的「與」&& 及「或」||

By Webb 2022-03-03

對非布林值進行「與」&& 及「或」|| 運算時

系統會先將其轉會為布林值後再進行運算，在返回原值

  • true && true

若兩個值經轉換後都為 true 則會返回後者

var a = 1 && 2;
console.log(typeof a); // number
console.log(a); // 2
  • true && false
  • false && true
  • false && false

若兩個值中有出現 false 則會返回第一個 false

var a = 3 && -8;
console.log(typeof a); // number
console.log(a); // -8

var a = 3 && 0;
console.log(typeof a); // number
console.log(a); // 0

var a = 0 && 3;
console.log(typeof a); // number
console.log(a); // 0

var a = 0 && 0;
console.log(typeof a); // number
console.log(a); // 0

var a = NaN && 0;
console.log(typeof a); // number
console.log(a); // NaN

var a = 0 && NaN;
console.log(typeof a); // number
console.log(a); // 0
  • true || true
  • true || false
  • false || true

若兩個值中有出現 true 則會返回第一個 true

var a = 3 || -8;
console.log(typeof a); // number
console.log(a); // 3

var a = 3 || 0;
console.log(typeof a); // number
console.log(a); // 3

var a = 0 || 3;
console.log(typeof a); // number
console.log(a); // 3

var a = NaN || 5;
console.log(typeof a); // number
console.log(a); // 5

var a = 6 || NaN;
console.log(typeof a); // number
console.log(a); // 6
  • false || false

若兩個值經轉換後都為 false 則會返回後者

var a = 0 || NaN;
console.log(typeof a); // number
console.log(a); // NaN

var a = NaN || 0;
console.log(typeof a); // number
console.log(a); // 0
